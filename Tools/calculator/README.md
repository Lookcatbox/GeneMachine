# Calculator

A simple browser-based calculator with a dark-themed UI. Supports addition (+), subtraction (−), multiplication (×), and division (÷), along with decimal input and a clear button (C). Operations are evaluated **left-to-right** as they are chained (e.g. `1 + 2 × 3` yields `9`, not `7`). Division by zero shows an error message in the display without crashing. Keyboard input is also supported: digits, `+`, `-`, `*`, `/`, `.`, `Enter`/`=`, `Backspace`, and `Escape`/`C`.

To open, either double-click `index.html` in your file explorer, or run `start index.html` from a Command Prompt / PowerShell window in this directory.
